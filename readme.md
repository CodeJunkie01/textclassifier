1. Create venv under ./environment
2. activate venv
3. pip install -r requirements.txt
4. create Zotero Library
5. create .env file

===== env vars =====
- OPENAI_SECRET
- OPENAI_ORG_ID
- ZOTERO_KEY
- ZOTERO_USER_ID
- ZOTERO_MAIN_COLLECTION_ID (optional; if not set user is asked for collection name on startup)
